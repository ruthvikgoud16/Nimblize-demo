## 📋 Conventions

- **Prompt templates** go in `assets/prompts/` as `.yaml` files
- **Screenshots** go in `screenshots/` with descriptive filenames
- **Diagrams** go in `diagrams/` as `.png` or `.svg` exports
- All changes are logged in `CHANGELOG.md` before merging
